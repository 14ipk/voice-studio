# VoiceForge — 声音克隆与语音合成工作台 PRD

## 1. 产品概述

VoiceForge 是一个面向内容创作者、播客主、配音爱好者及开发者的一站式声音克隆与语音合成 Web 应用。用户可上传自己的录音并转换为他人音色,可分析任意说话人语音并保存其"音色指纹"到云端音色库,可基于已克隆音色将任意文字合成为自然流畅的语音,并在最终导出前通过专业旋钮精细调节音高、语速、共振峰等全部声音参数。

- 解决问题:消除传统配音流程中"音色不可控、参数无法精细调节、克隆样本无法复用"的痛点。
- 目标用户:播客主、短视频创作者、有声书制作者、独立游戏开发者、AI 配音爱好者。
- 产品价值:将专业音频工作流中的"音色克隆 + TTS + 母带微调"三大能力整合到单一浏览器界面,降低门槛。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|---------|---------|
| 访客 | 无需注册 | 体验全部功能,音色库使用本地存储 |
| 注册用户 | 邮箱注册 | 云端持久化音色库、历史记录、参数预设管理 |

### 2.2 功能模块

1. **声音转换工作台 (Voice Convert)**: 上传录音 → 选择目标音色 → 保持原音调与语速,仅替换音色 → 实时预览
2. **音色克隆实验室 (Voice Clone Lab)**: 上传目标说话人样本(10-30秒) → 提取音色特征 → 命名并保存到音色库
3. **音色库管理 (Voice Library)**: 浏览/试听/重命名/删除已克隆音色,支持标签分类与搜索
4. **文字转语音合成 (TTS Studio)**: 输入文本 → 选择已克隆或预设音色 → 实时合成自然语音
5. **参数调节母带 (Mastering Knobs)**: 下载前通过专业旋钮调节音高、语速、共振峰、气息感、混响、亮度、压限等参数
6. **导出中心 (Export)**: 多格式(WAV/MP3)导出,采样率/位深可选

## 3. 核心流程

### 3.1 声音转换流程
用户进入工作台 → 切换到"声音转换"模式 → 上传或现场录制自己的语音 → 从音色库选择目标音色 → 点击"开始转换" → 系统保持原音调与语速,仅替换音色 → 进入母带调节面板微调 → 试听满意后导出下载。

### 3.2 音色克隆流程
用户进入"音色克隆实验室" → 上传目标说话人语音样本(可多段) → 系统提取音色特征并展示频谱可视化 → 用户命名并添加标签 → 保存到音色库 → 后续可在声音转换或 TTS 中复用。

### 3.3 TTS 合成流程
用户进入 TTS 工作室 → 输入或粘贴文本 → 选择目标音色(已克隆或预设) → 选择情感风格 → 点击"合成" → 进入母带调节 → 导出。

## 4. 用户界面设计

### 4.1 设计风格

**美学方向:暗夜音频实验室 (Neon Audio Lab)**
- 主色调:深炭黑底 (#0A0A0F) + 霓虹青绿 (#00F0B5) 主强调色 + 品红 (#FF2D7E) 次强调色
- 按钮风格:扁平+发光描边,悬停时霓虹辉光扩散
- 字体:Display 用 "Space Grotesk" 替代品 "Outfit"(几何无衬线,带科技感),正文用 "JetBrains Mono" 强调代码/数据感
- 布局风格:沉浸式全屏 + 浮动面板,左工具栏 + 中央画布 + 右属性栏的三栏专业工作台布局
- 图标风格:线性细描边图标(lucide-react),配合波形/频谱等动态数据可视化
- 装饰元素:背景流动的音频波形粒子、网格背景、毛玻璃面板

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 全局 | 顶部导航 | 半透明毛玻璃栏,Logo + 工作模式 Tab 切换 + 音色库按钮 |
| 声音转换页 | 上传区 | 虚线边框拖拽区,内含波形动画装饰,上传后显示真实波形 |
| 声音转换页 | 音色选择卡片 | 2x3 网格,每卡含迷你波形 + 试听按钮 + 选中高亮 |
| 音色克隆页 | 频谱可视化 | 实时绘制的频谱图、基频曲线、共振峰散点 |
| 音色库页 | 卡片网格 | 响应式网格,每卡含名称、标签、试听、操作菜单 |
| TTS 工作室 | 文本框 | 大型文本输入区,右侧实时字数与预计时长 |
| 母带面板 | 旋钮组 | 8 个圆形旋钮(可拖拽旋转),下方数值显示与重置按钮 |
| 导出页 | 格式选择 | 单选卡片(WAV/MP3)+ 采样率/位深下拉 |

### 4.3 响应式

- 桌面优先(≥1280px):完整三栏工作台
- 平板(768-1279px):右属性栏折叠为抽屉
- 移动端(<768px):单栏堆叠,工具栏移至底部 Tab
- 触摸优化:旋钮支持触摸拖拽,上传区支持点击选择

### 4.4 视觉氛围

- 全局背景:深炭黑 + 缓慢流动的霓虹波形粒子(canvas 动画)
- 面板:毛玻璃(backdrop-blur)+ 1px 半透明描边 + 微弱内发光
- 强调色辉光:所有可交互元素 hover 时产生青绿色或品红辉光
- 数据可视化:音色克隆时实时绘制频谱,合成过程显示动态波形
- 加载状态:霓虹脉冲扫描线,而非通用 spinner
